Tailwind Plug in Components

This section contains a simplified version of the components fully customizable with tailwind.

Buttons



        
  <!-- Primary -->        
  <button class="btn-blue-solid">Blue Solid</button>
  <!-- Secondary -->
  <button class="btn-blue-outline">Blue Outline</button>
  <!-- Tertiary -->
  <button class="btn-blue-transparent">Text Only</button>

Inputs


        
  <!-- Input -->   
  <input type="text" id="field" class="msc-input peer" placeholder="Input" />
  <!-- Label -->   
  <label for="field" class="msc-input-label" >Input label</label>

Drop downs


        
<select class="msc-dropdown">

  <option value="2">List item 2</option>
  <option value="3">List item 3</option>

</select>
